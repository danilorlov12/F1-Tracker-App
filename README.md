# F1-Tracker-App

The Android app that helps F1 fans view statistics about F1 drivers, teams, and races by specific year.

### Tech-stack:
* Kotlin
* Jetpack Compose
* ViewModel + StateFlow
* Kotlin Coroutines
* Modularisation
* JSON assets for offline data
* Retrofit + okhttp3
* Dagger + Hilt
* JUnit4
